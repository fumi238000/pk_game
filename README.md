# PK ゲーム

「サッカーの PK 戦」を、Ruby の基本的な機能で実装しました

## アプリ概要

サッカーの PK 戦を以下の流れで実装

1. ユーザーが蹴るコース・守るコースを選択
2. COM がランダムに蹴るコース・守るコースを選択
3. 結果の表示
4. 5 回ずつ行動した後,勝敗の表示

## 使い方

1. ターミナルで実行。アプリのディレクトリ上で`ruby main.rb`で実行
2. 画面のメッセージに従い、1,2,3 を選択

## インストール

```shell
$ git https://github.com/fumi238000/pk_game.git
$ cd pk_game
$ ruby main.rb
```
